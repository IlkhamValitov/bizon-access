Доступ к Bizon

Откуда брать API-токен

1. В Bizon не забывайте нажать сохранить "API-токен", не просто сгенерировать, а обязательно надо нажать СОХРАНИТЬ!

2. API Bizon работает через "Страницы регистрации" самого бизона - они нужны для смены даты, для регистрации пользователей в Bizon

Т.е. если нужна смена даты - в Bizon должна быть создана страница регистрации и к ней должна быть прикреплена комната.

Если нужно собирать базу в Bizon, также нужна страница регистрации, плюс подключена услуга "Страницы регистрации".
